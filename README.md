# 🛒 Amazon Clone (Frontend)

## 📌 Description
This project is a **frontend clone of Amazon** built using **HTML, CSS, and JavaScript**. It replicates key e-commerce functionalities such as product browsing, searching, cart management, and order tracking.

The project helped me gain hands-on experience in building a **complete frontend application** and integrating it with an **backend server avaialable online** to dynamically load product data and manage cart.

---

## 🛠 Tech Stack
- HTML5  
- CSS3  
- JavaScript  
- Backend API (for product data and cart)

---

## ✨ Features
- 🔍 Search products by **name or category**  
- 🛍 Browse products loaded from a backend server  
- ➕ Add products to cart directly from homepage  
- 🛒 Cart management system:
  - Update product quantity  
  - Delete items from cart  
  - Select delivery options
- 💾 Persistent cart and data handling  
- 📦 Orders page:
  - View past orders  
  - Track individual products  
- 🚚 Separate tracking page for each ordered product  
- 🎨 Clean and responsive UI  

---

## 🎯 What I Learned
- Building a **complete frontend application from scratch**  
- Handling dynamic data using APIs  
- DOM manipulation and event handling  
- Managing application state (cart, orders)  
- Designing multi-page workflows (Cart → Orders → Tracking)  
- Improving UI/UX with CSS
